Supported Endpoints
1) http://localhost:9103/interoperability/api/admin/healthcheck
Description: Checks connectivity with database
Parameters : None
2) http://localhost:9103/interoperability/api/admin/resetpasses
Description: Delete Passes
Parameters : None
3) http://localhost:9103/interoperability/api/admin/resetstations
Description: Initialise Stations with given data
Parameters : Given data
4) http://localhost:9103/interoperability/api/admin/resetvehicles
Description: Initialise Vehicles with given data
Parameters : Given data
5) http://localhost:9103/interoperability/api/PassesPerStation/:stationID/:date_from/:date_to
Description: Returns all data from Passes from specified station and between specified dates
Parameters : station ID + date from + date to
6) http://localhost:9103/interoperability/api/PassesAnalysis/:op1_ID/:op2_ID/:date_from/:date_to
Description: Returns all data from passes for op2_ID from stations that belong to op1_ID in given dates
Parameters : op1_ID + op2_ID + date from + date to
7) http://localhost:9103/interoperability/api/PassesCost/:op1_ID/:op2_ID/:date_from/:date_to
Description: Returns all data from Passes without passes list and total debt of op2 to op1
Parameters : op1_ID + op2_ID + date_from + date_to
8) http://localhost:9103/interoperability/api/ChargesBy/:op_ID/:date_from/:date_to
Description: Returns all data from Passes that belong to stations of given operator 1 for given dates
Parameters : op_ID + date_from + date_to
